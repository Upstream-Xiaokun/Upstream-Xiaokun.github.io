一、Java数据类型

编程语言分为：

    弱类型语言：JavaScript 、HTML、CSS

    强类型语言：Java 属于强类型语言，严格遵守编程的规范

八大数据类型（基本类型）

    整数类型：byte/short/int/long

    浮点类型：float/double

    字符类型：char

    布尔类型：boolean

引用类型（暂补充）

    String

    注意：String 不是关键字，String属于引用类型

什么是字节？

    字节是计算机用于存储容量的一种计量单位，作为单位来处理的最小二进制数字串


    一个字节有八个二进制为 例如：0100 1001

二、Java数据类型运用的注意事项

① 在运算精确的数值时，避免使用 float 和double 浮点类型，因为计算不精准，会出现问题
② 有关二进制、八进制、十六进制的区分
③ 字符的本质其实就是数字，所有的字符都是有一套数字编码表转换而来，例如ASCII码、Unicode
④ /t 、/n，属于转义字符
⑥ 对象和字符串常量池
    这里的字符串创建是放在字符串的常量池中，所以二只是相等的(true)
⑦ 布尔值只有 true 和 false 两种
⑧ 类型转换
    byte <char < short < int < long < float  < double  低类型——高类型

    由高——转换为低 自动转换

    由低——转换为高 强制转换

    布尔类型无法转换
⑨ JDK7新特性
    int i ＝ 100_0000_0000;

    创建较大的整形时，可以使用下划线来区分四个零

三、变量、常量、作用域

        变量：Java 中变量的定义 （数据类型 名字 ＝ 值）

变量

    类变量：加 static 关键字，属性可以直接使用

实例变量：从属于类对象，需要创建对象来进行使用，默认有初始化

    boolean 的默认值为 false

    除了基本类型，其余的默认值为 null

    局部变量：在方法之内，（注意：局部变量必须声明和初始化值，不然会报错，且无法使用）

常量

    final 常量名 ＝常量值

    用 final 修饰的参数为常量，常量的值一旦确定，就不能发生改变

作用域

包括四大类：public、protected、private、默认（容易把 protected 和 默认搞混淆）
    public：对于所有的类都可以访问

    private：只能在同一个 Class 中才能使用，作用域为类的内部

    protected：作用域为同一个包的所有类和外部包的子类。

    对于同一个包内的类，无论是是不是子孙类，都可以访问

    默认类型：默认类型的访问权限为同一个包内的所有类。

    外部包无论是不是子孙类都不能访问

Java的基本运算符

基本类型中需要注意的问题

① 在整形的计算中，输出的结果有小数的时候需要转换类型
    int a = 5;

    int b = 2;

    System.out.println(b/2);

    这里的实际结果输出应该是 2.5 但是输出的结果确实 2，应为整形不是浮点数，不存在小数之说

    进行一次类型转换，就正常了

    system.out.println(a/(double)b); // 这样输出的结果就正确了
② 在整形类型的运算中（longi/nt/byte/short/）
    没有 long 长整形的时候，所有非 int 的转化为 int

    int a = 10;

    btye b = 2;
